# Sum Positive Numbers

## 🧩 Problem

Write a function `sumPositive` that takes an array of numbers and returns the **sum of all positive numbers** in the array.

- Negative numbers should be ignored.
- Zero should not be included in the sum.

---

## 📥 Input

An array of numbers.

```javascript
sumPositive([1, -2, 3, 4, -5])
