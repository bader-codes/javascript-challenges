# Count Vowels

## 🧠 Problem

Write a function called `countVowels` that takes a string as input and returns the **number of vowels** in the string.

Vowels are:

```
a, e, i, o, u
```

The function should count vowels **regardless of letter case** (both uppercase and lowercase).

## ✨ Examples

### Example 1

```javascript
countVowels("javascript")
```

Output
3

### Example 2

```javascript
countVowels("Hello")
```

Output
2

### Example 3

```javascript
countVowels("sky")
```

Output
0
