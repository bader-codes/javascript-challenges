# Check if a Number Exists in an Array

## 🧩 Problem Description

Given an array of numbers and a target number, write a function that returns **true** if the target exists in the array, and **false** otherwise.

### Examples:

```javascript
Input: numbers = [1, 3, 5, 7], target = 7
Output: true

Input: numbers = [1, 3, 5, 7], target = 2
Output: false
