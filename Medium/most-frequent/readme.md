# Most Frequent Number - JavaScript Challenge

## Description

This repository contains a JavaScript challenge to find the number that appears most frequently in an array.  
It helps practice array manipulation, object usage, and loop logic.

---

## Problem

Write a function `mostFrequent(array)` that returns the number which occurs **most frequently** in the given array.  
If multiple numbers have the same highest frequency, return any one of them.

---

### Example

```js
const arr = [4, 1, 2, 2, 3, 3, 3, 4, 4, 4];
console.log(mostFrequent(arr)); // 4
