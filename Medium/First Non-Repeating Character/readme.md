# 🧠 First Non-Repeating Character

## 📌 Description
Write a function that takes a string and returns the **first non-repeating character**.  
If all characters are repeated, return `"null"`.

---

## 🚀 Examples

```js
firstNonRepeatingCharacter("aabbcdd") // "c"
firstNonRepeatingCharacter("aabbcc")  // "null"
firstNonRepeatingCharacter("abcabcde") // "d"
